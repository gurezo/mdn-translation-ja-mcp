# Editorial Guideline

## When to use

- MDN 翻訳の表記・約物・単位・カタカナ長音を確認するとき
- ブランディング・固有名詞の表記を決めるとき
- レビューで表記揺れを指摘するとき

## Quick checklist

- [ ] 日本語と英数字の間に半角スペース
- [ ] 日付・数字・記憶容量の表記がガイドライン準拠
- [ ] カタカナ長音ルール（-er/-or 等は長音、-eer/-ty 等は無長音）
- [ ] メニュー表記 `[項目]`（前後半角スペース、三点リーダー省略）
- [ ] 引用符「」、禁止記号（―、～）なし
- [ ] 用語表記が頻出語リストに一致

## Workflow

1. 翻訳対象の表記箇所を洗い出す
2. [references/editorial-guideline.md](references/editorial-guideline.md) で該当ルールを確認
3. 用語は mozilla-l10n-glossary スキルと併用
4. 文体は japanese-style スキルと併用
