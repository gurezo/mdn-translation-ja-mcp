## 基本文体

- 語尾は「です・ます調」
- 敬語（「～いたしました」「～くださいませ」等）は使用しない
- 同じ文章内・カテゴリー内で表記を統一
- 行頭に全角スペースは付けない

## 送りがな・ひらがna表記

- 動詞の送りがな: 読み込み、貼り付け（NG: 読込み、貼付け）
- 形式名詞はひらがな
- 接続詞・副詞はひらがな
- 読みづらい漢字は適宜ひらがな

## 表記統一（ひらがな推奨）

| NG | OK |
| --- | --- |
| 下さい | ください |
| 於いて、於ける | おいて、おける |
| 居る（補助的用法） | いる |
| 通り | どおり |
| 出来る | できる |
| の為 | のため |
| 毎（「ごとに」の意味） | ごと |
| 更に、稀に、直ぐに、全て、殆ど | さらに、まれに、すぐに、すべて、ほとんど |

## 漢字表記（推奨）

例えば、互いに、常に、時に、共に、仮に、必ず、先に、早速、再び、一つ

## 数量表記

- 「2か月」「5か所」「10か国語」
- 「すべて」はひらがな
- 「～することが～」はひらがな
- 「～することができます」→「～できます」
- 「～するとき」はひらがな、「～する場合」は漢字

## 文体の使い分け

- 「～のみ」: 文語調
- 「～だけ」: 口語調
- 「有り」「無し」: 単独使用時は漢字、それ以外はひらがな
- 「既に」: 漢字・ひらがな両方可
